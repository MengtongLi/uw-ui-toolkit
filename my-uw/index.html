These are the styles for the redesign of MyUW. You can view the beta redesign by logging in with your NetID here. This page is a work in progress. These styles are all namespaced under the class 'my-uw', which includes the styles for the frame as well (the header, search bar, sidebar, etc). All styles for the portlets themselves are namespaced under the class 'up-portlet-content-wrapper'.

Buttons

UW buttons build off the Bootstrap button styles. Buttons can be either button or a elements. Give every button a class of .btn .btn-uw, then add classes below to suit your needs.

Default buttons

Add .btn-lg, .btn-sm, or .btn-xs to change the size of your button.

Flat buttons

Add .btn-uw-flat for flat style. Note that the flat style cannot be combined with an action button class without manually adding another class.

Outline buttons

Add .btn-uw-outline for an outline button. Note that the outline style cannot be combined with an action button class without manually adding another class.

Action buttons

Add .btn-uw-dark, .btn-uw-success for form submission or success actions, .btn-uw-neutral for a neutral blue color, or .btn-uw-disabled for a button that should be disabled.
